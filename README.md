# Gestor de Asistentes - Eventos

## Descripción
Aplicación de consola en Java que permite gestionar el registro de asistentes a eventos utilizando un HashSet para evitar duplicados.

## Funcionalidades
- Registrar asistentes (nombre, apellido, email)
- Buscar asistente por email
- Mostrar número total de asistentes
- Listar todos los emails registrados

## Tecnologías
- Java
- HashSet
- Scanner
